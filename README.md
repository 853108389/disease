#### 基本介绍

韩国疫情随时间变化

文件： 

```
test.py 疫情可视化图

sir.py 疫情sir模型

by_xx.png 数据统计图

xx.csv 韩国疫情数据

xx.html 结果网页
```

要求：

python 2.6版本

环境： 

```
pip install matplotlib
pip install pyecharts
pip install pandas
```
